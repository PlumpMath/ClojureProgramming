    The Clojure REPL
	R - Read : 입력되어진 코드를 읽는다.
	E - Eval : 코드를 평가하고, 결과를 도출해낸다.
	P - Print : 결과를 출력한다.
	L - Loop : read 단계로 돌아간다.
